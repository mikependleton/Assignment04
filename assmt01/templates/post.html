1603 MSIEXEC error troubleshooting

How to get to the bottom of windows installer 1603?

My advice is to install Notepad ++. This is because Notepad or Mac's default text editors are:

  1. Open the installer log
  2. Ctrl + End for windows or Cmd + end for Mac
  3. Note the exit code
  4. If 1603 in this example, then ctrl + f and type "value 3"

    5. Reference for value types:https://docs.microsoft.com/en-us/windows/win32/msi/logging-of-action-return-values

  5. Once you search then click on the top hit
  6. Scroll up about 5-10 lines and then you will see the reason for the failure

    7. This is data from Work, so I did need to scrub our data for security reasons. {% load static %}

  7. This particular instance there is a permissions issue and the systems team would need to investigate GPO/permissions.
